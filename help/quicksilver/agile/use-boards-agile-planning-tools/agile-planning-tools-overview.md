# 敏捷規劃工具概述

當您使用臨機卡和連接卡建立展示板時，您可能希望利用其他靈活的規劃工具，包括列WIP限制、將卡片分組到泳道中，以及可用於積壓工作的進紙欄。

您可以以適合您和您的團隊的方式使用任何或所有這些工具，而無需提交特定的靈活方法，如看板或掃描。
