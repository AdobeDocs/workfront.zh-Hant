# 在專案上使用Agile

您可以針對您的專案利用敏捷功能，而不會遇到敏捷實務通常伴隨的管理挑戰（例如管理團隊待辦專案、建立反複專案等）。 如需詳細資訊，請參閱[在[!UICONTROL 敏捷]檢視](../../manage-work/projects/manage-projects/manage-projects-in-agile-view.md)中管理專案。
