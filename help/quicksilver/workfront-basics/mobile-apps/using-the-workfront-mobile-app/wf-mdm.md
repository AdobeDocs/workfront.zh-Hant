# [!DNL Adobe Workfront] for [!DNL Mobile Device Management] (MDM)

Adobe [!DNL Workfront] 與一些 [!DNL Mobile Device Management] (MDM)平台，為組織提供更安全、更受控制的移動體驗。 此 [!DNL Workfront] 行動應用程式適用於下列行動裝置管理平台：

## MobileIron

請參閱 [!DNL MobileIron]安裝的說明檔案 [!DNL Workfront] 行動應用程式：

* iOS
* Android

有關配置的資訊 [!DNL Workfront] for [!DNL MobileIron]，請參閱 [設定 [!DNL Adobe Workfront] for [!DNL MobileIron]](../../../workfront-basics/mobile-apps/using-the-workfront-mobile-app/wf-mobileiron-configs.md).
