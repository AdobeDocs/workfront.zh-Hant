# 變更[!DNL Adobe Workfront]中日期的格式

<!--this article used to be called "Change the date format in Adobe Workfront when using Chrome". The team decided to make it more generic and hide the steps. Also see drafted content below-->

<!--Remove me October 2026-->

>[!IMPORTANT]
>
>此頁面上說明的程式已移除，因為它僅適用於尚未上線至Adobe Admin Console的組織。
>
>所有Workfront組織現在都已上線Adobe Admin Console。

您的日期偏好設定是由Adobe Unified Shell中設定的語言偏好設定所控制。 預設語言設定（因此也是預設日期設定）是`en-US`。

如需詳細資訊，請參閱 [!DNL Workfront]](/help/quicksilver/workfront-basics/navigate-workfront/workfront-navigation/adobe-unified-experience.md/#access-your-profile-and-preferences)的[[!DNL Adobe Unified Experience] 。
