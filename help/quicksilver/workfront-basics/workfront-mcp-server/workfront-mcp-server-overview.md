# Adobe Workfront MCP伺服器概述

[!DNL Adobe Workfront] MCP （模型內容通訊協定）伺服器可讓您在支援的AI代理平台上，透過自然語言對話處理Workfront資料。 您不需要按一下Workfront來尋找專案、更新任務或檢查核准，而是會以簡單的英文詢問AI代理平台，而AI代理平台會為您工作。

您不需要知道Workfront API或有關MCP的任何資訊即可使用Workfront MCP伺服器。 設定AI代理平台與Workfront通話一次，然後開始詢問。

## 設定和使用Workfront MCP伺服器

在您可以使用Workfront MCP伺服器之前，Workfront管理員必須在Workfront中啟用MCP伺服器存取，並且您必須將您的AI代理平台連線到Workfront。 如需詳細資訊，請參閱[設定Adobe Workfront MCP伺服器](/help/quicksilver/workfront-basics/workfront-mcp-server/configure-workfront-mcp-server.md)。

設定後，您可以要求AI代理平台尋找、建立、更新和管理Workfront專案。 如需可用動作、範例要求及資料處理的詳細資訊，請參閱[使用Adobe Workfront MCP伺服器](/help/quicksilver/workfront-basics/workfront-mcp-server/use-workfront-mcp-server.md)。
