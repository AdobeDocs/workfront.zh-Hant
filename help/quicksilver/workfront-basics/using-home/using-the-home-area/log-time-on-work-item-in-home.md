# 從[!UICONTROL 首頁]區域登入工作專案的時間

您可以在「首頁」區域中「工作清單」中的任何工作專案上記錄時間。 如需詳細資訊，請參閱[記錄時間](/help/quicksilver/timesheets/create-and-manage-timesheets/log-time.md)。
