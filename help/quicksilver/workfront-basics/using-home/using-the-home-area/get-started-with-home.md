# 開始使用[!UICONTROL 首頁]

<!--Audited: 12/2023-->

本文說明[!DNL Adobe Workfront]中目前[!UICONTROL 住家]區域的概觀。 如需有關使用新[!UICONTROL 首頁]的資訊，請參閱[開始使用新首頁](../new-home/get-started-with-new-home.md)。

有兩種方式可在目前的[!UICONTROL 首頁]區域檢視您的工作：

* 從[!UICONTROL 工作清單]
* 從[!UICONTROL 行事曆]。

## [!UICONTROL 工作清單]

您可以在[!UICONTROL 工作清單]的單一位置檢視所有指派的工作：

![[!UICONTROL 工作清單]面板](assets/worklist-and-right-panel-home.png)

檢閱目前[!UICONTROL 首頁]區域中的下列區段（如上圖醒目提示）：

* **A**：使用左側面板來篩選、排序及選取指派給您的工作。

* **B**：在左側面板中選取工作專案後，請使用右側面板與自訂欄位互動、進行更新並記錄時間。

如需使用[!UICONTROL 首頁工作清單]的詳細資訊，請參閱下列文章：

* [在[!UICONTROL 首頁]區域的[!UICONTROL 工作清單]中顯示專案](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md)
* [從[!UICONTROL 首頁]區域建立工作專案](../../../workfront-basics/using-home/using-the-home-area/create-work-items-in-home.md)

## [!UICONTROL 行事曆]

您可以顯示工作專案到期的時間，以視覺化您的工作，也可以在[!UICONTROL 首頁]區域的[!UICONTROL 行事曆]中檢閱您的工作，以封鎖完成工作的時間：

![](assets/calendar-home-nwe-350x114.png)

檢閱目前[!UICONTROL 首頁]區域中的下列區段（如上圖醒目提示）：

* **A**：使用[!UICONTROL 行事曆]檢視建立您需要完成的工作的視覺化地圖。 只要按一下工作專案，然後從[!UICONTROL 工作清單]拖曳到[!UICONTROL 行事曆]以封鎖工作的逾時時間。

* **B**：使用[!UICONTROL 行事曆]的同步選項來與您的[!DNL Outlook]行事曆同步。 在一個地方管理會議和工作。

* **C**：使用[!UICONTROL 詳細資料]按鈕開啟新的右側面板，您可以在此檢視工作專案的詳細資訊。

* **D**：使用到期日列來追蹤指派給您的工作何時到期。

如需如何使用首頁行事曆的詳細資訊，請參閱[使用[!UICONTROL 首頁行事曆]檢視](../../../workfront-basics/using-home/using-the-home-area/use-home-calendar-view.md)。

## 尋找您的工作

[!UICONTROL 首頁]區域是您尋找指派給您的任務、問題和核准的一站式服務。 這些篩選、分組和排序選項可共同運作，以支援您以您想要的方式組織工作專案。 您可以使用分組來分組專案，使用篩選器來專注於需要完成的工作，最後使用排序來按分組的升序或降序列出專案。

>[!NOTE]
>
>篩選和排序選項會儲存在瀏覽器中。 如果您一律在同一部電腦上使用相同的瀏覽器（且不清除網站資料），篩選和排序將不會變更，但如果您切換瀏覽器或電腦，則篩選和排序將會不同。

### 將類似的工作專案與分組一起分組

您可以使用分組在工作清單的通用標題下顯示相似的專案。 您可以依下列方式群組專案：

* [!UICONTROL 計畫完成]
* [!UICONTROL 計劃開始]
* [!UICONTROL 認可日期]
* [!UICONTROL 專案]
* [!UICONTROL 我的優先順序]

如需如何使用群組的詳細資訊，請參閱[!UICONTROL 首頁]區域](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md)的[!UICONTROL 工作清單]中的[顯示專案。

![](assets/group-by-drop-down-expanded-in-home-with-planned-start-date-nwe-350x273.png)

### 使用篩選器縮小您的焦點

[!UICONTROL 工作清單]篩選器可讓您將焦點縮小至特定工作專案。

以下範例說明如何篩選在「首頁」中檢視的專案：

* 依專案型別和狀態篩選。

  例如，如果您想要顯示所有任務，可以選取[!UICONTROL 任務]篩選器。 如果您希望比此專案更明確，並且只顯示已準備好要您開始處理的任務，請在「[!UICONTROL 任務]」篩選下選取「[!UICONTROL 已準備好開始]」篩選。

* 僅依專案型別篩選。

  例如，您可以選取[!UICONTROL 問題]以檢視所有狀態的所有問題（[!UICONTROL 處理]或[!UICONTROL 已要求]），或選取[!UICONTROL 核准]以檢視所有工作專案、存取權要求、時程表、檔案和校訂核准。

如需如何使用篩選器的詳細資訊，請參閱[!UICONTROL 首頁]區域](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md)的[!UICONTROL 工作清單]中的[顯示專案。

![](assets/displaying-work-items-filters-nwe-350x401.png)

### 存取指派給您團隊的工作

[!UICONTROL 首頁]區域具有永久分組（[!UICONTROL 團隊請求]），可顯示指派給團隊的任務和問題。 團隊會顯示在此區段中。 按一下團隊名稱會開啟該團隊的「團隊請求」頁面。

>[!NOTE]
>
>篩選和排序選項不會影響[!UICONTROL 團隊請求]群組。 只要您已將工作指派給團隊，就可以看到此分組。

如需存取團隊請求的詳細資訊，請參閱[!UICONTROL 首頁]區域](../../../workfront-basics/using-home/using-the-home-area/manage-work-and-team-requests-home.md)中的文章[[!UICONTROL 管理]工作和團隊請求。

![](assets/team-requests-expanded-home-group-by-drop-down-nwe-350x314.png)

### 監視您提交的工作

追蹤您直接從[!UICONTROL 工作清單]提交核准的工作。 當您選取檢視[!UICONTROL 工作清單]中的[!UICONTROL 核准]時，我已提交的[!UICONTROL 核准有永久群組]。 您可以提醒核准者需要核准的工作。 如有需要，您也可以恢復核准。

>[!NOTE]
>
>篩選和排序不會影響我已提交的[!UICONTROL 核准]群組。 只要您有工作等待核准，就可以看到此分組。
>
>[!UICONTROL 核准]篩選器中的核准數未顯示您提交核准的專案數。 [!UICONTROL 核准]篩選器只會顯示等待您核准的專案。

![](assets/approvals-expanded-home-group-by-drop-down-nwe-350x415.png)

## 優先處理對您而言重要的事項

[!UICONTROL 工作清單]可讓您透過[!UICONTROL 我的優先順序]群組，呈現對您而言重要的工作並排定其優先順序。 您最多可以新增20個專案至[!UICONTROL 我的優先順序]清單。

>[!NOTE]
>
>任何新增至[!UICONTROL 我的優先順序]的工作專案只會顯示給您，沒有人可以為您排定工作優先順序。

如需如何使用[!UICONTROL 我的優先順序]的詳細資訊，請參閱[優先處理[!UICONTROL 住家]區域](../../../workfront-basics/using-home/using-the-home-area/prioritize-work-in-home.md)的工作。

![](assets/prioritizing-in-home-group-by-drop-down-nwe-350x412.png)

## 完成它

按一下[!UICONTROL 工作清單]中的專案名稱之後，請使用[!UICONTROL 首頁]中的右面板，更新並完成您的工作。

### 在為您自訂的空間中工作

您的[!DNL Workfront]管理員可以新增最多16個自訂欄位到[!UICONTROL 工作清單]中的專案，以便更輕鬆地尋找和更新您需要的資訊。 大部分欄位都可編輯，因此您不必導覽至工作專案。

若要更新此區域中的任何欄位，只要按一下欄位即可。 如果欄位可編輯，您可以在右側面板中修改它。

>[!NOTE]
>
>若將此欄位從配置範本新增至工作專案，則無法更新[!UICONTROL 認可日期]。

![](assets/16-fields-home-350x137.png)

### 以重要資訊更新其他人

與[!UICONTROL 住家]區域的其他人快速有效率地通訊。 使用工作專案的註解區域，您可以提出問題、通知其他人最近的更新，或回覆來自[!UICONTROL 首頁]區域的直接註解。 此外，您還可以透過下列方式，向其他人更新您目前的進度：

* 正在更新註解區域中的[!UICONTROL 認可日期]

  >[!NOTE]
  >
  >   這是您唯一可以在首頁更新認可日期的位置。

* 新增註解或更新專案的[!UICONTROL 條件]，報告您對事情的感受
* 更新完成百分比列

如需在[!UICONTROL 首頁]區域通訊的詳細資訊，請參閱[更新工作](../../../workfront-basics/updating-work-items-and-viewing-updates/update-work.md)。

![](assets/updates-home-350x198.png)

### 記錄您的時間

追蹤您處理個別工作專案的時數。 您可以選擇一般時數型別，例如[!UICONTROL 任務時間]，或者您可以選擇由[!DNL Workfront]管理員設定的專案特定時數型別。

如需有關如何在[!UICONTROL 首頁]區域記錄時間的詳細資訊，請參閱文章[記錄時間](../../../timesheets/create-and-manage-timesheets/log-time.md)中的[首頁](../../../timesheets/create-and-manage-timesheets/log-time.md#home)一節。

![](assets/log-time-home-350x181.png)
