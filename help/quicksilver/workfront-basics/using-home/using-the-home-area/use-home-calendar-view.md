# 使用 [!UICONTROL 首頁日曆] 檢視

您可以在 [!UICONTROL 首頁] 管理工作日和工作分配的區域。 只有您可以檢視和管理 [!UICONTROL 首頁日曆].

>[!NOTE]
>
>此 [!UICONTROL 日曆] 檢視 [!UICONTROL 首頁] 區域與 [!DNL Adobe Workfront] [!UICONTROL 日曆] 報告，此動態報告可提供工作的視覺表示。 如需 [!DNL Workfront] [!UICONTROL 日曆] 報表，請參閱 [日曆報表概觀](../../../reports-and-dashboards/reports/calendars/calendar-reports-overview.md).

## 排程您的 [!UICONTROL 首頁日曆]

您可以使用 [!UICONTROL 首頁日曆] 排程您一週的工作項目。

>[!NOTE]
>
>將工作項目放在 [!UICONTROL 首頁日曆] 不會更改與工作項關聯的現有日期。

新增 [!DNL Workfront] 工作項到日曆中：

1. 前往 [!UICONTROL 首頁日曆] 視圖，如 [檢視 [!UICONTROL 首頁日曆]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).

   指派給您或您的群組、團隊或工作角色的所有工作項目都會顯示在左側面板中。 您可以排序清單並篩選清單，以僅顯示特定項目。

   如需詳細資訊，請參閱 [篩選和排序工作項目](#filter-and-sort-work-items) 稍後的文章。

1. 從左側面板，將工作項拖動到要處理它的日期和時間。

   如果您計畫在項目計畫完成日期之後或超過項目計畫小時數的時間內，則工作項目將以紅色顯示，並出現警告。

   要了解工作項目到期日及其如何影響 [!UICONTROL 首頁日曆]，請參閱  [[!UICONTROL 首頁日曆] 檢視](../../../workfront-basics/using-home/using-the-home-area/home-calendar-view.md).

1. 執行下列任一操作：

   * **要將工作項計畫多天，請執行以下操作：** 重複步驟2以在您想要處理項目時，每天和每次新增項目。
   * **要修改日曆項的持續時間，請執行以下操作：** 將滑鼠移到項目的底邊上，直到游標變成雙箭頭，然後將底邊拖動到所需的結束時間。
   * **要刪除日曆項，請執行以下操作：** 暫留在項目上，然後按一下 [!UICONTROL 特拉斯坎] 表徵圖。

## 更新您的 [!UICONTROL 首頁日曆]

您可以更新顯示在 [!UICONTROL 首頁日曆] 檢視。 您的更新也會顯示在 [!UICONTROL 更新] 頁簽。

1. 前往 [!UICONTROL 首頁日曆] 視圖，如 [檢視 [!UICONTROL 首頁日曆]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).

1. 開啟 [!UICONTROL 詳細資料] 要更新的工作項的面板：

   * 從左側面板：選擇工作項，然後按一下 **[!UICONTROL 詳細資料]**.
   * 從日曆：按一下排程的日曆事件。

   工作項的詳細資訊顯示在窗口右側的面板中。

   ![](assets/click-cal-item-to-see-details-350x217.png)

1. 執行下列任一操作以更新工作項：

   * 在 **[!UICONTROL 開始新更新]** 框中，鍵入更新，指定以下任何資訊，然後按一下 **[!UICONTROL 更新]**.

   ![](assets/refresh-qs-350x360.png)

   當您開啟 [!UICONTROL 首頁日曆] 檢視或重新整理 [!UICONTROL 首頁日曆].

## 導覽至不同的周

1. 前往 [!UICONTROL 首頁日曆] 視圖，如 [檢視 [!UICONTROL 首頁日曆]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).
1. 在 **[!UICONTROL 日曆工具列]**，按一下日期指標的左箭頭，將日曆檢視移回一週，或按一下右箭頭，前移一週。

   ![](assets/week-arrows-350x206.png)

   您可以按一下 **[!UICONTROL 今天]** 返回當周。

## 指定您檢視的日曆

您可以指定要在您的PTO、生日或假日日曆上查看的整合日曆，如 [!UICONTROL 首頁日曆]. 如需指示，請參閱 [設定您的 [!UICONTROL 首頁日曆] 檢視設定](../../../workfront-basics/using-home/using-the-home-area/configure-home-calendar-view.md).
