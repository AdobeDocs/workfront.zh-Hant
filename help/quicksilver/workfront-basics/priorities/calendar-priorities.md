# 行事曆

<span class="preview">此頁面上的資訊是指尚未普遍提供的功能。 它只能在「預覽Sandbox」環境中使用。</span>

行事曆檢視現在可在「優先順序」中使用

透過清楚的視覺月曆，輕鬆追蹤您的工作。 使用「優先順序」行事曆，您可以

* 使用篩選器尋找您的工作
* 顯示狀態和焦點層級等欄位
* 套用顏色以快速優先順序
及更多內容！
