# 建立請求

<!--
{{highlighted-preview}}
-->

請求和問題在Adobe Workfront中可互換。 您可以執行下列任一項作業來建立請求：

* 在Workfront中提交請求
* 在其他應用程式中提交請求
* 透過電子郵件將請求傳送到已設定為請求佇列的專案。

所有請求會顯示在其已提交專案的問題區域中。
