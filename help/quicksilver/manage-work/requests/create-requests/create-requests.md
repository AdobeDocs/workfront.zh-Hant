# 建立請求

<!--
{{highlighted-preview}}
-->

請求和問題可在Adobe Workfront中互換。 您可以執行下列其中一項操作來建立請求：

* 在Workfront中提交請求
* 在其他應用程式中提交請求
* 將請求透過電子郵件傳送至已設定為「請求佇列」的專案。

所有請求都會顯示在其已提交專案的「問題」區域中。
