# 刪除請求草稿

您可以在建立請求草稿時刪除請求草稿，方法是將其捨棄。

如需詳細資訊，請參閱：

* [建立及提交Adobe Workfront請求](../../../manage-work/requests/create-requests/create-submit-requests.md)
* [從草稿建立請求](../../../manage-work/requests/create-requests/create-requests-from-drafts.md)

如果您不再認為草稿相關，則可在草稿請求另存為草稿後刪除這些請求。 您無法復原已刪除的草稿請求。

## 刪除請求草稿的先決條件

您必須先執行下列操作，才能刪除請求草稿：

* 開始建立請求。 如此會在「草稿」區段中自動將請求儲存為草稿。

   如需建立請求的詳細資訊，請參閱 [建立及提交Adobe Workfront請求](../../../manage-work/requests/create-requests/create-submit-requests.md).

## 刪除請求草稿

1. 按一下 **主菜單** 圖示 ![](assets/main-menu-icon.png) 在Adobe Workfront的右上角。
1. 按一下 **請求**，然後按一下 **草稿** 中。

   所有請求佇列的所有草稿都會顯示在此清單中。

1. （選用）按一下 **依請求類型篩選** 在草稿清單的右上角，選取包含您要顯示之草稿的請求佇列。
1. 在清單中選取草稿，然後按一下 **刪除** 清單頂端。
1. 按一下 **是，刪除它**.

   草稿被刪除，無法恢復。
