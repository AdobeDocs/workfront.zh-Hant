# 問題資訊

處理專案時，您可能會發現發生非預期的事件。 您可以將這些意外事件記錄為特定專案或任務的問題。

與任務不同，問題無法推動項目的完成率。 不過，問題是無法完成專案的工作項目。 未完成的問題表示在項目關閉之前仍有工作要完成。
