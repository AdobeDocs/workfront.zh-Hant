# 問題資訊

處理專案時，您可能會發現發生未預期的事件。 您可以將這些非預期事件記錄為特定專案或任務的問題。

與任務不同，問題不能驅動專案的完成百分比。 但是，問題指可能會阻止專案完成的工作專案。 未完成的問題表示專案關閉前仍有工作要完成。
