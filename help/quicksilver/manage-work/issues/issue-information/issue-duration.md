# 問題持續時間概觀

問題的持續時間是問題的計畫起始日期和計畫完成日期之間的天數。 

問題持續時間由Adobe Workfront自動計算，無法手動修改。 

您可以在問題報告或問題清單的檢視中顯示問題的持續時間。 

如需建立報表的詳細資訊，請參閱 [建立自訂報表](../../../reports-and-dashboards/reports/creating-and-managing-reports/create-custom-report.md).
