# 更新任務優先順序

您可以定義任務的優先順序，以指明任務的重要性。

您必須具有任務的「管理」權限，才能修改任務的「優先順序」。

## 任務優先順序概覽

使用任務優先順序時，請考慮以下事項：

* Adobe Workfront管理員可以修改Workfront中的優先順序標籤，也可以建立新的優先順序。 因此，您在Workfront執行個體中的「優先順序」欄位可能有的選項，可能會與下列選項不同。 如需自訂執行個體中優先順序名稱的相關資訊，請參閱 [建立和自訂優先順序](../../../administration-and-setup/customize-workfront/creating-custom-status-and-priority-labels/create-customize-priorities.md).
* 每個優先順序標籤都有一個與其關聯的編號，無法修改。

   您應熟悉組織中使用之優先順序的編號順序。

   例如，如果已修改優先順序的標籤，您的組織可能會使用數字1來指代緊急任務。

   請參閱下表以了解每個預設優先順序標籤及其對應編號：

   | **優先順序標籤（可修改）** | **優先順序編號（無法修改）** |
   |---|---|
   | 無 | 0 |
   | 低 | 1 |
   | 標準 | 2 |
   | 高 | 3 |
   | 緊急 | 4 |



## 修改任務的優先順序

1. 轉到要修改其優先順序的任務。
1. 按一下 **任務詳細資訊** 中。
1. 按一下 **編輯** 圖示 ![](assets/qs-edit-icon.png)，然後在 **概述** 區域中，為 **優先順序** 欄位。
1. 按一下 **儲存** **變更**.
