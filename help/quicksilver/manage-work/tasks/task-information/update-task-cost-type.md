# 更新任務成本類型

任務的計畫成本和實際成本及其人工成本由每項任務的成本類型確定。

您可以為專案內的個別任務配置成本類型。 每個成本類型都會影響計畫成本值和實際成本值。

如需在Adobe Workfront中追蹤成本的詳細資訊，請參閱 [追蹤成本](../../../manage-work/projects/project-finances/track-costs.md).

## 配置單個任務的成本類型

1. 轉到要配置成本類型的任務。
1. 按一下 **任務詳細資訊** 在左側面板中，展開 **金融** 的上界。
1. 按兩下 **成本類型** 並選擇要應用於任務的成本類型。

   ![](assets/task-cost-type-options-in-tasks-details-nwe-350x233.png)

   從下列選項中選取：

   * 無成本
   * 固定小時
   * 使用者小時
   * 角色小時

   有關每個任務成本類型的詳細資訊，請參閱 [追蹤成本](../../../manage-work/projects/project-finances/track-costs.md).

1. 按一下 **儲存** **變更** **.**
