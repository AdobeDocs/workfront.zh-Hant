# 恢復已刪除的項目

需要還原專案時，請連絡您的Adobe Workfront管理員或群組管理員。 它們可以還原過去30天內已刪除的項目、任務、問題和其他項目。 30天後，項目會永久刪除且無法還原。

如需還原專案的詳細資訊，請參閱 [還原已刪除的項](../../../administration-and-setup/manage-workfront/manage-deleted-items/restore-deleted-items.md).
