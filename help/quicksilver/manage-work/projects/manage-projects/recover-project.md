# 復原已刪除的專案

需要還原專案時，請聯絡Adobe Workfront管理員或群組管理員。 他們可以還原過去30天內已刪除的專案、任務、問題和其他專案。 30天後，專案會永久刪除且無法還原。

如需有關還原專案的詳細資訊，請參閱[還原已刪除的專案](../../../administration-and-setup/manage-workfront/manage-deleted-items/restore-deleted-items.md)。
