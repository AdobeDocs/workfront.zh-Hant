## 設定專案的條件類型

1. 轉到要更新條件類型的項目。
1. 執行下列任一操作： 

   * 按一下 **更多** 功能表 ![](assets/qs-more-menu.png) 在專案名稱的右側，按一下 **編輯**.
   * 按一下 **專案詳細資料** 中。

   ![](assets/update-condition-type-nwe-350x108.png)

1. 在 **條件類型** 欄位中，選擇以下選項之一：

   * **手動：** 專案擁有者會手動設定專案的條件。

      在這種情況下，項目所有者可以更新項目標題或項目詳細資訊部分中的項目條件。

   * **進度狀態：** Workfront會根據專案的進度狀態設定條件。 有關如何計算進度狀態的資訊，請參見 [項目進度狀態概觀](../../../manage-work/projects/planning-a-project/project-progress-status.md).

1. 按一下 **儲存** 修改「編輯項目」(Edit Project)框中的「條件類型」(Condition Type)時。

   按一下 **儲存變更** 修改「項目詳細資訊」部分中的「條件類型」時。
