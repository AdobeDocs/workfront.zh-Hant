## 設定專案的條件型別

1. 前往您要更新其條件型別的專案。
1. 執行下列其中一項：

   * 按一下專案名稱右側的&#x200B;**更多**&#x200B;功能表![更多功能表](assets/qs-more-menu.png)，然後按一下&#x200B;**編輯**。
   * 按一下左側面板中的&#x200B;**專案詳細資料**。

   ![](assets/update-condition-type-nwe-350x108.png)

1. 在&#x200B;**條件型別**&#x200B;欄位中，選擇下列其中一項：

   * **手動：**&#x200B;專案擁有者手動設定專案的條件。

     在這種情況下，專案所有者可以在專案標題或專案詳細資訊區段中更新專案狀況。

   * **進度狀態：** Workfront會根據專案的進度狀態設定條件。 如需如何計算進度狀態的詳細資訊，請參閱[專案進度狀態概觀](../../../manage-work/projects/planning-a-project/project-progress-status.md)。

1. 在編輯專案方塊中修改條件型別時，按一下&#x200B;**儲存**。

   在專案詳細資訊區段中修改狀態型別時，按一下&#x200B;**儲存變更**。
