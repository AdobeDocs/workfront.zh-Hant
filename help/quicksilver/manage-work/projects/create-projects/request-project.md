# 要求專案

您可以在開始使用專案前，先請求核准。 這會建立狀態為Idea的專案。 我們建議為您的項目構建一個業務案例，以概述通過完成該項目希望實現的一些關鍵點，並向您的管理人員演示您的人工和費用預算。

在Adobe Workfront的下列區域新增專案時，可以請求專案：

* 在「項目」區域中
* 在Portfolio的「項目」區域中
* 在方案的「項目」區域中
* 在項目組的「組」區域中，當您是組管理員時。

## 要求專案

1. 執行下列任一項作業：

   * 按一下 **主菜單** ![](assets/main-menu-icon.png)，按一下 **專案**，然後展開 **新增專案**.
   * 前往產品組合，然後展開 **新增專案**.
   * 前往程式，然後展開 **新增專案**.
   * 如果您是群組管理員，也可以在您管理之群組的「專案」區段中建立專案。 如需詳細資訊，請參閱 [建立和修改群組的專案](../../../administration-and-setup/manage-groups/work-with-group-objects/create-and-modify-a-groups-projects.md).

1. 按一下 **要求專案**.

   ![](assets/new-project-dropdown-nwe-350x358.png)

1. 執行下列任一項作業：

   * 定義業務案例時，請遵循與建立專案相同的步驟。 （建議）

      有關定義業務案例和請求項目的資訊，請參閱 [為項目建立業務案例](../../../manage-work/projects/define-a-business-case/create-business-case.md).

   * 從頭建立專案或使用範本，並將狀態變更為 **已請求**.

      如需建立專案的相關資訊，請參閱 [建立專案](../../../manage-work/projects/create-projects/create-project.md) 或 [使用範本建立專案](../../../manage-work/projects/create-projects/create-project-from-template.md).
   您請求的專案狀態為 **已請求** 它會顯示在 **請求的項目** 標籤(如果您將其與Portfolio關聯)。 產品組合經理現在可以審核它，然後批准或拒絕它，在這種情況下，項目的狀態將更改為 **已核准** 或 **已拒絕**.
