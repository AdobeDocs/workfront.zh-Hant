# 項目計畫收益概覽

計畫福利是指如果項目完成，組織可能從中獲益的估計金額。

計畫福利不由Adobe Workfront計算。 作為項目責任人，您可以在業務案例中人工指定項目的計畫福利。 我們建議每個組織建立如何確定此值的政策。

項目的計畫效益會影響項目的淨值。\
有關計算項目淨值的詳細資訊，請參閱文章 [計算淨值](../../../manage-work/projects/project-finances/calculate-net-value.md).

## 找出項目的計畫優勢

您可以在Workfront的以下區域查看或更新項目的計畫效益：

* 在「業務案例摘要」中。\
   有關項目的「業務案例」的詳細資訊，請參閱文章 [為項目建立業務案例](../../../manage-work/projects/define-a-business-case/create-business-case.md).

* 在項目報表中，將「計畫福利」列添加到報表的視圖時。\
   如需在Workfront中建立報表的詳細資訊，請參閱文章 [建立自訂報表](../../../reports-and-dashboards/reports/creating-and-managing-reports/create-custom-report.md).

* 在Portfolio優化程式中。\
   如需有關Portfolio優化程式的詳細資訊，請參閱文章 [Portfolio優化程式概述](../../../manage-work/portfolios/portfolio-optimizer/portfolio-optimizer-overview.md).

## 更新項目的計畫效益

1. 轉到要更新計畫福利的項目。
1. 按一下 **業務案例** 中。
1. 按一下 **編輯專案資訊** 並在 **計畫福利** 欄位。

1. 按一下&#x200B;**儲存**。
