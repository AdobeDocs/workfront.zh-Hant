# 將專案新增至產品組合

建議您在起始專案時，將專案新增至產品組合。 不過，您可以在有生之年隨時將其新增至產品組合。

將專案新增至產品組合時，請考量下列事項：

* 雖然產品組合可能包含不限數量的專案，但您只能將一個產品組合與專案建立關聯。
* 專案會保留在產品組合中，直到移除或與其他產品組合建立關聯為止。

## 將專案新增至產品組合

1. 前往產品組合，然後按一下 **[!UICONTROL 專案]** 中。

   ![](assets/qs-portfolio-with-projects-350x90.png)

1. 按一下 **[!UICONTROL 新增專案]** 並選擇添加項目的方法。

   >[!TIP]
   >
   >檢視中的專案清單時，無法新增專案 [!UICONTROL 里程碑] 檢視。

   從下列選項中選取：

   <table style="table-layout:auto"> 
    <col> 
    <col> 
    <tbody> 
     <tr> 
      <td role="rowheader">[!UICONTROL現有項目]</td> 
      <td> <p>新增已建立的專案。</p> </td> 
     </tr> 
     <tr> 
      <td role="rowheader">[!UICONTROL新項目]</td> 
      <td> <p>新增專案。 </p> <p>如需建立新專案的詳細資訊，請參閱 <a href="../../../manage-work/projects/create-projects/create-project.md" class="MCXref xref">建立專案</a>. </p> </td> 
     </tr> 
     <tr> 
      <td role="rowheader">[!UICONTROL從導入項目 [!DNL MS Project]] </td> 
      <td> <p>新增您先前從匯出的專案 [!DNL MS Project] 並儲存在電腦上。 </p> <p>有關從導入新項目以建立新項目的詳細資訊 [!DNL Microsoft Project]，請參閱 <a href="../../../manage-work/projects/create-projects/import-project-from-ms-project.md" class="MCXref xref">從匯入專案 [!DNL Microsoft Project]</a>.</p> </td> 
     </tr> 
     <tr> 
      <td role="rowheader">[!UICONTROL請求項目]</td> 
      <td> <p>請求核准專案。</p> <p>如需請求專案的相關資訊，請參閱 <a href="../../../manage-work/projects/create-projects/request-project.md">請求專案</a>. </p> </td> 
     </tr> 
     <tr> 
      <td role="rowheader">[!UICONTROL從模板中新增]</td> 
      <td> <p>使用現有範本新增專案。 </p> <p>如需從範本建立專案的詳細資訊，請參閱 <a href="../../../manage-work/projects/create-projects/create-project-from-template.md" class="MCXref xref">使用範本建立專案</a>.</p> </td> 
     </tr> 
    </tbody> 
   </table>

   ![](assets/new-project-dropdown-expanded-from-portfolio-nwe-350x376.png)

1. （條件性）如果您選取新增現有專案，請開始在 **[!UICONTROL 新增專案]** 框中，按一下它們。 您一次可以新增數個專案。

   ![](assets/add-existing-project-from-portfolio-window-nwe-350x545.png)

1. 按一下 **[!UICONTROL 新增專案]**.

   您選取的專案或專案現在會與產品組合相關聯。
