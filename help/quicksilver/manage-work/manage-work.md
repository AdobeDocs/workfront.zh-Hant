# 管理工作資訊

您可以使用Adobe Workfront來管理您的工作。

您可以先根據工作的複雜性以及您配置給工作的資源或預算，決定使用哪些工作物件來定義您的工作。

本節說明如何將您的工作組織成較大的工作專案，例如專案、投資組合或方案，以及如何將較大的工作分割成更易於管理的工作專案，例如任務和問題。

此外，您可以使用甘特圖來定義您工作的時間表，或在無法完成工作時委派給其他人。
