# 已移除舊版校對檢視器

>[!IMPORTANT]
>
>本文說明獨立產品中的功能 [!DNL Workfront Proof]. 有關內部校對的資訊 [!DNL Adobe Workfront]，請參閱 [校對](../../../review-and-approve-work/proofing/proofing.md).

2018.3版之前，支援舊版校對檢視器。 2018.3版（2018年11月）中，舊版校樣檢視器和所有依賴Flash的其他應用程式已移除，不再提供使用。
