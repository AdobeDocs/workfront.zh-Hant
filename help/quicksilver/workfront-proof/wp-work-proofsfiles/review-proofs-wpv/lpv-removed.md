# 舊版校訂檢視器已移除

>[!IMPORTANT]
>
>本文提及獨立產品[!DNL Workfront Proof]中的功能。 有關[!DNL Adobe Workfront]內部校訂的資訊，請參閱[校訂](../../../review-and-approve-work/proofing/proofing.md)。

在2018.3版之前，已支援舊版校訂檢視器。 在2018.3版（於2018年11月）中，舊版校訂檢視器和依賴Flash的所有其他應用程式已移除，不再提供。
