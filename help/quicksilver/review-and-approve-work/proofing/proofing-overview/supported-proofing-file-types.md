# 支援的校對檔案類型和大小限制概觀

即使審核者的電腦上未安裝用於建立檔案的軟體(如Photoshop或Illustrator)，您仍可以與他們共用和審核各種類型的檔案。 我們支援靜態、視訊、音訊、網頁擷取和互動式檔案類型，大小最多4 GB。

## 檔案大小限制

Workfront有下列限制：

* 每個檔案的檔案必須小於4 GB。
* XLS和XLSX檔案必須小於100 MB。
* 附加至校樣註解的檔案必須小於30 MB。
* PNG的高度和寬度必須小於65535px。
* 靜態合併校樣最多可支援50個檔案。
* 上傳為校樣的檔案必須少於2,000頁。

>[!NOTE]
>
>較大的檔案上傳所花的時間較長，如果連線不穩定，動作可能會終止/中斷。

## Zip檔案

基於安全原因，我們不支援或從ZIP檔案中解壓檔案。 先解壓縮檔案再上傳。 若未執行此動作，上傳將不會失敗。

只有郵遞區號檔案包含完整的互動式專案，且您的帳戶已啟用互動式功能（Premium計畫）時，才可提交。

如需詳細資訊，請參閱 [互動式內容校樣概觀](../../../review-and-approve-work/proofing/proofing-overview/interactive-content-proofs.md).

## InDesign和誇克檔案

我們不支援InDesign和誇克檔案，因為它們不是自包含的檔案類型，它們引用了許多其他檔案，如影像和字型。 這些檔案類型的輸出會根據輸出設定而有所不同。 建議客戶將InDesign和Quark檔案儲存為PDF並上傳PDF。
