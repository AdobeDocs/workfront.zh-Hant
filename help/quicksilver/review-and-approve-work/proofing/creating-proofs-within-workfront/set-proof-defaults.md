# 設定個人校訂預設值

您可以定義個人校訂預設設定，以套用至您建立的校訂。 每次您在Workfront中產生首次校訂或上傳新校訂版本時，都會套用這些預設值。

## 設定個人校訂預設值

{{step1-to-proofing}}

1. 在右上角，按一下您的虛擬人偶，然後選取&#x200B;**個人設定**。
1. 選擇&#x200B;**校訂預設值**&#x200B;標籤，然後指定下列資訊：
