# 檔案決定狀態概觀

您可以直接在檔案清單中檢視檔案的狀態：

![檔案清單](assets/status-in-doc-list.png)中的狀態

                            </li>

                                <p>其他核准者可能已作出「已核准變更」或「已核准」的決定
                            </li>
                        </ul>
