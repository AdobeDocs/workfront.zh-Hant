# 2026年第一季度報告增強功能

本頁說明了2026年第一季度版本對「預覽」環境所做的報告增強功能。 如上所述，這些增強功能將於生產環境中提供。

如需2026年第一季發行週期目前可用的所有變更清單，請參閱[2026年第一季發行概觀](/help/quicksilver/product-announcements/product-releases/26-q1-release-activity/26-q1-release-overview.md)。

## 在畫布控制面板中複製報告

>[!NOTE]
>
>預覽版本： 2025年10月23日
>所有客戶的生產： 2025年10月23日
>[!BADGE 不在排程]{type=Neutral}內

KPI、表格或圖表報表建立後，您現在可以在畫布控制面板中複製該報表。 複製後，您可在儲存前視需要編輯報表。

## 從報告篩選器移除欄位選項

>[!NOTE]
>
>預覽： 2025年11月6日
>生產快速發行： 2025年11月13日
>適用於所有人的生產： 2026年1月15日

我們已移除下列先前在套用篩選器至報表時可使用的欄位選項：

* 其他群組識別碼
* 其他角色ID
* 其他團隊ID

由於與「不等於」和「為空白」運運算元相關的問題，已移除這些專案。 如果您有使用這些欄位之一的現有篩選器，它將繼續如期運作。 或者，您可以在<code>文字模式中繼續使用這些欄位</code>，但建議避免在操作時使用「不等於」或「為空」運運算元。

下列欄位選項可作為替代選項：

* 其他群組： ID
* 其他角色： ID
* 其他團隊： ID

## 改善畫布儀表板中群組計數的顯示

>[!NOTE]
>
>預覽： 2025年11月6日
>生產快速發行： 2025年11月13日
>適用於所有人的生產： 2026年1月15日

當表格報告有多個結果頁面且表格設定了分組時，表格現在會顯示目前頁面的記錄金額以及所有頁面的整體記錄計數。 例如，如果您的表格報表有7個群組，且第一頁顯示為3，則表格將顯示第3頁（共7頁）。


## 新護欄可縮短畫布儀表板中的載入時間

>[!NOTE]
>
>預覽： 2025年11月6日
>生產快速發行： 2025年11月13日
>適用於所有人的生產： 2026年1月15日

為避免載入時間延遲並改善畫布儀表板中的整體效能，我們已對可以新增到儀表板的儀表板元件數量套用了限制：

* 每個控制面板的報告：25個限制
* 表格檢視的分組：5個限制
* 與報表基本物件的距離： 10個限制
* 表格檢視上的欄： 25限制
* 儀表板層級篩選提示： 10個限制
