# 2019.3發行活動概觀

本頁提供2019.3版中包含功能的相關資訊。 本頁面上的功能已於2019年8月19日當周在生產環境中使用。 如需發行時間的特定日期和時間說明，請參閱 [Workfront信任網站](https://trust.workfront.com/).

## 所有使用者的增強功能

2019.3版包含下列增強功能：

### 專案增強功能

* [「開啟」篩選器現在會在「問題」清單中顯示更多結果](../../../../product-announcements/product-releases/quarterly-release-archive/2019.3-release-activity/2019.3-project-enhancements.md#changes)

### 全新Workfront體驗測試版現已可在預覽中提供 {#the-new-workfront-experience-beta-now-available-in-preview}

除了新功能外，新的Workfront體驗還提供全新的現代設計。

### Workfront程式庫現已推出 {#workfront-library-now-available}

Workfront資料庫是企業數位內容中心，可用來管理組織數位資產的完整生命週期。

### 不再支援API第3版 {#api-version-3-no-longer-supported}

API第3版支援已於2018年11月終止。 如需各版Workfront API支援排程的詳細資訊，請參閱 [API版本設定與支援排程](../../../../wf-api/api/api-version-support-schedule.md)

### TLS 1.0和1.1支援即將結束 {#tls-1-0-and-1-1-support-is-ending}

為提供最佳安全性，Workfront要求所有使用TLS 1.1或更舊版本的整合和網路流量都升級為使用TLS 1.2。在預覽環境中，TLS 1.0已停用。
