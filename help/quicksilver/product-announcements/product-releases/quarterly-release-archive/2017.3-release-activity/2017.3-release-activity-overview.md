## 行動裝置增強功能 {#mobile-enhancements}

>[!NOTE]
本節所述的功能現在可搭配您的Workfront生產帳戶使用；這不是預覽功能。

2017年8月，Android和iOS的Workfront行動應用程式生產版本新增了下列功能：

* 提交請求
* 工時單新條目
* 自訂表單編輯
* 校樣核准請求
* 訂閱項目的推播通知

如需這些變更的詳細資訊，請參閱 [更新iOS和Android適用的行動應用程式（2017年8月初）](../../../announcements/announcement-archive/updated-mobile-app.md).

在11月8日當周的生產版本之前，我們將提供下列額外功能：

* 存取專案清單
* 訪問項目上的任務和子任務的清單
* 存取專案的問題清單
* 記錄專案的新問題
