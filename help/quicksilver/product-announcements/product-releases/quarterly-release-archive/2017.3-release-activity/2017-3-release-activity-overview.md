# 2017.3版本活動概覽

本頁提供2017.3版所含功能的相關資訊。 

此頁面上說明的功能目前可在預覽環境中使用，並將於11月在生產環境中使用。  

>[!IMPORTANT]
>

2017.3版包含適用於Workfront管理員和其他使用者的增強功能。

## 行動裝置增強功能 {#mobile-enhancements}

>[!NOTE]
>
>本節所述的功能現在可與您的Workfront生產帳戶搭配使用；這不是預覽功能。

下列功能已於2017年8月新增至Android和iOS適用的Workfront行動應用程式生產版本：

* 提交請求
* 時程表新專案
* 自訂表單編輯
* 校訂核准請求
* 訂閱專案的推播通知

如需這些變更的詳細資訊，請參閱[iOS和Android適用的更新行動應用程式（2017年8月初）](../../../announcements/announcement-archive/updated-mobile-app.md)。

下列附加功能將於11月8日當週的生產版本發行前提供：

* 存取您的專案清單
* 存取專案中的任務和子任務清單
* 存取專案的問題清單
* 在專案上記錄新問題
