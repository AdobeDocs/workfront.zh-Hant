# 2019.1發行活動概觀

本頁提供2019.1版所包含功能的相關資訊。

>[!IMPORTANT]
>
> 在發佈至生產環境之前，預覽環境中的功能可能會有所變更。 如果您正在查看此文檔的PDF版本，請訪問聯機版本以獲取最新資訊（從此文檔中的連結轉到聯機版本）。

2019.1版包含管理員和其他使用者的增強功能。

## 管理員的增強功能

* [從資源回收筒還原模板](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#restore)

   Workfront管理員現在可以從資源回收筒還原範本。

* [顯示首頁日期欄位的時間戳記](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#show)

   Workfront管理員現在可以在「工作清單」和「行事歷」中使用「配置範本」，選擇顯示或隱藏到期日的時間戳記。

* [為開啟校樣的非收件者設定預設校樣角色](https://support.workfront.com/hc/en-us/articles/360016372633#configure-default-proofing-role)

   Workfront管理員現在可為非校樣工作流程中指定的收件者，但透過其Workfront物件（例如專案、工作或問題）存取校樣的使用者設定預設校樣角色。

* [「項目首選項」下可用的所有持續時間類型](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#all)

   在「設定」的「項目首選項」區域中設定預設任務和問題持續時間類型時，Workfront管理員可以使用「簡單」、「工作驅動」、「計算分配」和「計算工作」選項。

## 所有使用者的增強功能

### 專案增強功能

* [敏捷改進](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#agile)

   通過按單個用戶或按小組篩選看板或清單板，查看您需要的資訊。

* [將任務和問題從清單添加到小版本](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#add)

   您現在可以使用任務或問題清單、報表或控制面板，將動態新增至小版本。 您也可以在一個迭代中為多個團隊指派動態。

* [甘特圖導出的新紙張大小](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#new)

   您現在可以將甘特圖打印在A1和A2紙張大小上。

* [在編輯模式下刪除對甘特圖中對話框的訪問](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#removed)

   當甘特圖處於編輯模式時，您無法再訪問「高級分配」對話框。 在「任務清單甘特圖」處於編輯模式時，只能進行內聯編輯。

* [在項目清單甘特圖中查看項目群或Portfolio中的任務資訊](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#view)

   您現在可以在方案或Portfolio的「項目清單甘特圖」中查看有關項目任務的資訊。

* [在模板上訪問任務清單甘特圖](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#access)

   您現在可以在項目模板中訪問任務清單甘特圖。

* [重新命名甘特圖上的專案檢視](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#renamed)

   我們將甘特圖上的「專案」檢視選項重新命名為「全部調整」。

* [任務清單甘特圖中的假設方案](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#what-if)

   現在，當任務在甘特圖的編輯模式中顯示時，可以對項目中的任務執行各種操作。

* [任務清單改進](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#task)

   在18.3測試版第4版本進行簡短移除後，重新設計的任務清單已重新啟用。 您現在也可以使用各種額外功能。

* [全螢幕顯示清單](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#display)

   當專案或工作清單大於畫面大小時，現在當您捲動時，清單會自動顯示在整個瀏覽器視窗中。

* [其他區域的新清單](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#new2)

   我們提高了效能，並改善了項目和任務清單在其他領域的外觀和風格。

* [以XLSX格式傳送傳送的報表](https://support.workfront.com/hc/en-us/articles/360016246394#send-a-delivered-report)

   除了所有其他目前的格式，您現在可以排程要以MS Excel(.xlsx)格式傳送報表。

* [導出審核日誌](https://support.workfront.com/hc/en-us/articles/360016246394#export-audit-logs)

   您現在可以將稽核記錄項目匯出至CSV檔案。 您一次最多可以將50,000個稽核記錄項目匯出至CSV檔案。

* [以滑鼠右鍵按一下「任務編號」欄，將無法再使用任務清單上的WBS檢視](https://support.workfront.com/hc/en-us/articles/360016246394#wbs-view-from-the-task-list)

   您無法再以滑鼠右鍵按一下任務清單中的「任務編號」欄，以顯示專案的「工作劃分結構」(WBS)。

### 資源管理增強功能

* [更新資源計畫器中的預設篩選器](https://support.workfront.com/hc/en-us/articles/360016247594#updated-default-filter)

   資源計畫器中的預設篩選器不再按項目組進行篩選。

* [為資源規劃器篩選器使用通配符](https://support.workfront.com/hc/en-us/articles/360016247594#using-wildcards-for-resource-planner-filters)

   現在，在資源規劃器中建立篩選器時，可以使用通配符。

* [支援資源規劃器中基於日期的通配符篩選變數](https://support.workfront.com/hc/en-us/articles/360016247594#support-for-date-based-wildcard-filter-vatiables)

   在資源規劃器中建立篩選器時，您現在可以使用任何版本的$$TODAY通配符篩選器變數。

* [導出資源計畫員中的角色視圖的選項](https://support.workfront.com/hc/en-us/articles/360016247594#export-options-for-the-role-view)

   您現在可以在「職責」視圖中從資源計畫器中選擇要導出的資訊層。

* [用於導出資源規劃器的資料格式選項](https://support.workfront.com/hc/en-us/articles/360016247594#data-formatting-options-for-exporting-the-resource-planner)

   現在，從資源計畫器導出時，可以選擇在Excel檔案中顯示資訊的方式。

* [永續性排程時間表](https://support.workfront.com/hc/en-us/articles/360016247594#persistent-scheduling-timeline)

   現在當您重新整理時間軸或離開頁面導覽時，排程時間軸會保留時間軸和篩選器。

* [資源計畫器中的新導出選項](https://support.workfront.com/hc/en-us/articles/360016247594#new-export-options-in-the-resource-planner)

   您現在可以從資源計畫員中選擇要導出的資訊層。

* [更新至資源計畫員中的用戶視圖](https://support.workfront.com/hc/en-us/articles/360016247594#update-to-the-user-view-in-the-resource-planner)

   當您未將任何基於用戶的篩選器應用到資源計畫器時，系統中的所有用戶現在都將顯示在資源計畫器的「用戶視圖」中。

### 校對增強功能

* [校對網頁校對檢視器中的互動式內容](https://support.workfront.com/hc/en-us/articles/360016372633#proof-interactive-content-in-the-web-proofing-viewer)

   如果貴組織的安全性原則不允許使用獨立式案頭測試檢視器應用程式，您的Workfront管理員現在可以在網頁測試檢視器中啟用互動式內容。

* [校對檢視器中留言的預設排序順序現在為最舊至最新](https://support.workfront.com/hc/en-us/articles/360016372633#default-sorting-order-for-comments)

   在校對檢視器中，校樣上留言的預設排序順序現在是「最舊」到「最新」，如同口頭對話中一樣。

* [增強與視訊範圍相關聯的校對檢視器中的留言檢閱功能](https://support.workfront.com/hc/en-us/articles/360016372633#enhanced-reviewing-for-comments)

   當您在校對查看器中查看與一系列視頻素材關聯的注釋時，您現在可以按一下「播放」來查看整個素材範圍。

* [從校樣通知或校樣檢視器連結至檔案詳細資訊](https://support.workfront.com/hc/en-us/articles/360016372633#link-to-document-details)

   當您收到電子郵件邀請您檢閱校樣，或您正在校樣檢視器中檢閱校樣時，現在可以快速存取校樣的「檔案詳細資訊」頁面。

* [在校對檢視器中變更您的電子郵件通知](https://support.workfront.com/hc/en-us/articles/360016372633#change-your-email-notifications)

   現在，所有校對審核者都可以指定他們要接收哪些校樣通知。 這在與外部利益攸關方合作時尤其重要。

* [更改案頭校對查看器中的背景顏色](https://support.workfront.com/hc/en-us/articles/360016372633#change-the-background-color-in-the-desktop-proofing-viewer)

   現在，您可以將案頭校對檢視器的背景顏色，從預設的近黑色變更為白色。

* [從校樣清除快取的瀏覽器資料](https://support.workfront.com/hc/en-us/articles/360016372633#clear-cached-browser-data-from-a-proof)

   現在，您可以清除可能隨校樣儲存的瀏覽器快取資料，讓所有內容都出現在案頭打樣檢視器中，審核者可以查看並加上註解。

### 行動裝置增強功能

* [全新行動搜尋體驗](https://support.workfront.com/hc/en-us/articles/360016374093#new-mobile-search-experience)

   現在，在Workfront行動應用程式中搜尋項目更方便使用者使用，且更符合直覺。

* [首頁是Workfront行動應用程式上新的預設登陸頁面](https://support.workfront.com/hc/en-us/articles/360016374093#home-is-the-new-default)

   首頁區域現在是Workfront行動應用程式的新預設登陸頁面。 iOS和Android裝置均支援此功能。

* [在Workfront行動應用程式中組織清單的新選項](https://support.workfront.com/hc/en-us/articles/360016374093#new-options-for-organizing)

   您現在可以在Workfront行動應用程式中篩選、排序和排序清單。

* [支援適用於Workfront裝置的iOS測試版應用程式](https://support.workfront.com/hc/en-us/articles/360016374093#support-for-workfront-beta-app-for-ios)

   您現在可以註冊成為iOS裝置上Workfront行動應用程式的測試版測試者。

* [為Android測試版建立簡單任務](https://support.workfront.com/hc/en-us/articles/360016374093#simple-task-creation-for-android-beta)

   您現在可以直接從專案、工作或從Android行動應用程式測試版的首頁區域建立工作。

* [適用於Android裝置的Workfront行動應用程式全新首次教學課程](https://support.workfront.com/hc/en-us/articles/360016374093#new-first-time-tutorial)

   更新第一次在Android裝置上存取Workfront行動應用程式時所顯示的教學課程外觀。

### 協力廠商應用程式整合增強功能

* [Workfront改善Slack](https://support.workfront.com/hc/en-us/articles/360016374093#workfront-for-slack-improvements)

   現在，Workfront提供多種更新供Slack，包括可在Slack中張貼您對Workfront中物件的留言，以便對物件留言，訂閱當某人對您的其中一項請求留言時，可在Slack中收到通知等。

* [Workfront支援Jira增強功能](https://support.workfront.com/hc/en-us/articles/360016374093#workfront-for-jira-enhancements)

   Jira適用的Workfront整合現在包含各種增強功能，包括指出整合何時在Jira中自動建立使用者、停用自動建立Workfront使用者的選項，以及執行其他更多作業。

2018年12月13日，生產環境直接提供以下Agile工具的更新：

* 在「看板積壓」頁簽上篩選和分組物料
* 查看小版本上任務和問題的父資訊

### API 10版現已可預覽 {#api-version-10-now-available-in-preview}

API 10版現在可在預覽環境中使用。 18.3版將可在生產環境中使用。 如需詳細資訊，請參閱 [API 10版的新功能。](https://support.workfront.com/hc/en-us/articles/360010455333) 

### 要刪除的各種單一登入選項 {#various-single-sign-on-options-to-be-removed}

下列單一登入(SSO)選項現在僅支援有限的容量；將於2019年下半年從產品中移除：

* LDAP\
   如需如何將LDAP與Workfront搭配使用的詳細資訊，請參閱 [使用LDAP設定Workfront。](https://support.workfront.com/hc/en-us/articles/217223517)

* 主動式目錄\
   有關將Active Directory與Workfront一起使用的詳細資訊，請參見 [使用Active Directory配置Workfront。](https://support.workfront.com/hc/en-us/articles/217223547)

* SAML 1.1\
   如需有關搭配Workfront使用SAML 1.1的詳細資訊，請參閱 [使用SAML 1.1設定Workfront。](https://support.workfront.com/hc/en-us/articles/217194887)

以下是這些SSO選項的刪除時間軸：

* **有限支援：** 2018年8月至2019年1月\
   重大問題將得到解決；不會解決嚴重性低的問題。

* **淘汰：** 2019年1月至2019年8月\
   不支援。

* **Workfront不再提供：** 2019年8月

這些變更正在為增強的驗證體驗做準備，目前只有有限數量的客戶才能使用。 如需這些近期變更的詳細資訊，請參閱 [增強驗證體驗快速入門。](https://support.workfront.com/hc/en-us/articles/360008329033)
