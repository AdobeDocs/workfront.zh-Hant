# 2019.2發行活動概觀

本頁提供2019.2版所含功能的相關資訊。 若要以PDF格式下載此內容，請前往 [此處](https://documentation.my.workfront.com/library/a/6626eb71-16b2-4460-bd81-9adcd4953eb5).

>[!NOTE]
>
>2019.2版已於下列日期在生產環境中提供：
>
>* *Workfront叢集2、4和6的生產環境 — 5月23日*
>* *Workfront叢集1、3和5的生產環境 — 5月24日*
>* *Workfront校樣生產環境 — 5月21日*


2019.2版包含系統管理員和其他使用者適用的增強功能。

## 管理員的增強功能

* [指定小時條目上的作業角色](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-administrator-enhancements.md)

   您現在可以在Workfront中記錄時間時指定工作角色。

* [允許沒有校對授權的使用者套用動作和解析註解](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-proofing-enhancements.md)

   沒有校對授權的使用者現在可以套用動作至留言並解決留言。

## 所有使用者的增強功能

### 資源管理增強功能

* [新的資源管理訪問級別](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-resource-management-enhancements.md#new)

   現在， Resource Management提供了新的訪問級別，允許具有計畫許可證的用戶編輯訪問。


* [按資源計畫員中的Portfolio優先順序排序項目](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-resource-management-enhancements.md#order)

   您現在可以在資源計畫器中按項目組合優先順序排序項目。

* [資源規劃器篩選器改進](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-resource-management-enhancements.md#resource)

   現在，當您將新欄位（組）添加到內置篩選條件時，資源計畫器中的篩選體驗更簡單。

* [在資源計畫器中顯示Portfolio優先順序](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-resource-management-enhancements.md#resource)

   您現在可以在「資源規劃器」中啟用一個設定，該設定允許您根據項目分配給的產品組合顯示項目的優先順序。

* [在資源計畫器中共用篩選器](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-resource-management-enhancements.md#share)

   您現在可以與選定的用戶、團隊、組、職務角色或公司共用在資源計畫器中建立的篩選器。

### 敏捷增強功能

* [在敏捷工具中追蹤問題的完成百分比進度](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-agile-enhancements.md#track)

   您現在可以追蹤看板和看板上問題的進度。

### 專案增強功能

* [在任務清單中以粗體顯示父任務的整行](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-project-enhancements.md#display)

   在任務清單中，包含父任務的行以粗體顯示。

* [新清單中的新欄寬預設值](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-project-enhancements.md#new)

   現在，Workfront會根據每欄中的值格式資訊自動調整欄的寬度。

* [撤消對任務清單的更改](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-project-enhancements.md#reverse)


   任務清單上的新「自動保存」(Autosave)按鈕允許您選擇是否要自動保存所做的更改，或者是否要在保存更改之前查看更改的效果。

* [將預設的自訂Forms和核准程式附加至工作](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-project-enhancements.md#attach)

   您現在可以設定預設的自訂表單和核准程式，以在將工作新增至專案時附加至工作。

* [自訂狀態時更快找到群組](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-project-enhancements.md#find)

   「設定」區域中「狀態」標籤上的下拉式功能表現在是「預先輸入」功能表。

### 整合增強功能

* [更新Workfront以整合Adobe Creative Cloud](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-project-enhancements.md#updated)

   Workfront for Adobe Creative Cloud擴充功能現在包含與Workfront資料庫的整合。

* [Workfront for Microsoft Teams改善](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-integration-enhancements.md#workfron)

   現在當您將Workfront用於Microsoft Teams時，可以提交工作請求並管理Workfront通知。

### 校對增強功能

* [允許沒有校對授權的使用者套用動作和解析註解](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-proofing-enhancements.md#allow)

   沒有校對授權的使用者現在可以套用動作至留言並解決留言。

* [在校對檢視器中註解校樣時標籤任何使用者](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-proofing-enhancements.md#tag)

   當您對校樣發表評論時，現在可以標籤或「@mention」任何具有Workfront所辨識之電子郵件地址的人。

### 行動裝置增強功能

* 改善iOS和Android中的清單體驗

   所有清單檢視的外觀和風格現在已在iOS和Android應用程式中更新。

## 直接發行至生產環境的增強功能


* 在任何「校樣核准」報表上，您現在可以使用新的「是目前的檔案版本」篩選條件，僅包含等待您核准的最新版本校樣。


* 如果您有的對象已不再使用，您現在可以停用它們以在清單中隱藏它們，以防止用戶將它們與其他對象關聯。

* [自訂狀態時更快找到群組](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-project-enhancements.md#find)

   「設定」區域中「狀態」標籤上的下拉式功能表現在是「預先輸入」功能表。

### Workfront程式庫現已推出 {#workfront-library-now-available}

Workfront資料庫是企業數位內容中心，可用來管理組織數位資產的完整生命週期。

### 不再支援API第3版 {#api-version-3-no-longer-supported}

API第3版支援已於2018年11月終止。 如需各版Workfront API支援排程的詳細資訊，請參閱 [API版本設定與支援排程](../../../../wf-api/api/api-version-support-schedule.md).

### TLS支援即將終止 {#tls-support-is-ending}

為提供最佳安全性，Workfront要求所有使用TLS 1.0或更舊版本的整合和網頁流量均須升級為使用TLS 1.1或更新版本。 此變更已在預覽環境中生效。

如需詳細資訊，請參閱 [Adobe Workfront中需要TLS 1.2](../../../../product-announcements/announcements/announcement-archive/tls-1-disabled.md).

### Flash應用程式移除 {#flash-application-removal}

自2018.3版本以來，仰賴Flash技術的所有功能已從Workfront中移除，並更換為新解決方案。

有關舊版Flash工具及其替換項的詳細資訊，請參見 [取代Adobe Workfront中的Flash型工具](../../../../product-announcements/announcements/announcement-archive/replace-flash-tools.md).

### 要刪除的各種單一登入選項 {#various-single-sign-on-options-to-be-removed}

下列單一登入(SSO)選項現在僅支援有限的容量；將於2019年下半年從產品中移除：

* LDAP

   如需如何將LDAP與Workfront搭配使用的詳細資訊，請參閱 [使用LDAP設定Adobe Workfront](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* 主動式目錄

   有關將Active Directory與Workfront一起使用的詳細資訊，請參見 [使用Active Directory配置Adobe Workfront](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* SAML 1.1

   如需有關搭配Workfront使用SAML 1.1的詳細資訊，請參閱 [使用SAML 1.1設定Adobe Workfront](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

以下是這些SSO選項的刪除時間軸：

* 有限支援: 2018 年 8 月 - 2019 年 1 月

   在此期間提交的主要問題將得到解決；不會解決嚴重性低的問題。

* 淘汰：2019年1月至2019年8月

   不支援。

* Workfront不再提供：2019年8月
