### API 10版現已可預覽 {#api-version-10-now-available-in-preview}

API 10版現在可在預覽環境中使用。 18.3版將可在生產環境中使用。 如需詳細資訊，請參閱 [API 10版的新功能](../../../../wf-api/api/new-api-version-10.md). 

### 不再支援API第3版 {#api-version-3-no-longer-supported}

API第3版支援將於2018年11月終止。 如需各版Workfront API支援排程的詳細資訊，請參閱 [API版本設定與支援排程](../../../../wf-api/api/api-version-support-schedule.md).

### TLS支援即將終止 {#tls-support-is-ending}

為提供最佳安全性，Workfront要求所有使用TLS 1.0或更舊版本的整合和網頁流量均須升級為使用TLS 1.1或更新版本。 此變更已在預覽環境中生效。

如需詳細資訊，請參閱  [Adobe Workfront中需要TLS 1.2](../../../../product-announcements/announcements/announcement-archive/tls-1-disabled.md).

### JSONP支援即將終止 {#jsonp-support-is-ending}

由於JSONP（含邊框間距的Javascript）是舊標準，具有已知的安全性弱點，自2018年11月起，Workfront將不再支援JSONP。 如需詳細資訊，請參閱 [終止支援JSONP](../../../../wf-api/api/ending-support-jsonp.md).

### Flash應用程式移除 {#flash-application-removal}

所有依賴Flash技術的功能預計於2018.3版本中從Workfront中移除，並更換為新解決方案。

有關舊版Flash工具及其替換項的詳細資訊，請參見 [取代Adobe Workfront中的Flash型工具](../../../../product-announcements/announcements/announcement-archive/replace-flash-tools.md).

### 要刪除的各種單一登入選項 {#various-single-sign-on-options-to-be-removed}

下列單一登入(SSO)選項現在僅支援有限的容量；將於2019年下半年從產品中移除：

* LDAP

   如需如何將LDAP與Workfront搭配使用的詳細資訊，請參閱 [使用LDAP設定Adobe Workfront](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* 主動式目錄

   有關將Active Directory與Workfront一起使用的詳細資訊，請參見 [使用Active Directory配置Adobe Workfront](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* SAML 1.1

   如需有關搭配Workfront使用SAML 1.1的詳細資訊，請參閱 [使用SAML 1.1設定Adobe Workfront](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

以下是這些SSO選項的刪除時間軸：

* **有限支援：** 2018年8月至2019年1月

   重大問題將得到解決；不會解決嚴重性低的問題。

* **淘汰：** 2019年1月至2019年8月

   不支援。

* **Workfront不再提供：** 2019年8月

這些變更正在為增強的驗證體驗做準備，目前只有有限數量的客戶才能使用。

### 舊版Workfront Fusion更新 — 2018年9月20日 {#legacy-workfront-fusionupdate-september-20-2018}

舊版Workfront Fusion是整合平台，可讓您將Workfront連線至多個協力廠商應用程式。

透過最新更新，舊版Workfront Fusion現在提供改良的設計，讓建立新FLO的程式更直覺。 具體變更包括：

* **新流量：** 按一下新的FLO現在會將使用者帶往「設計人員」畫面，而非先提示選取連接器。 此變更可協助您了解整體FLO，並了解任何FLO都應從事件開始。
* **添加函式：** 函式選擇器已更新，以便更容易找到並選取所需的函式。 更新了函式的說明以及輸入和輸出欄位，以新增直觀的指引。 新的「最受歡迎」區段可協助您開始使用最常使用的函式。
