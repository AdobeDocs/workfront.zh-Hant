# 23.1發佈概述

本頁提供了23.1版中包含的功能的相關資訊。

這些增強功能已在2023年1月18日和19日發佈的23.1版中提供。

23.1版網路研討會於2023年1月10日星期二舉行。 你可以 [註冊參加網路研討會，在此查看按需錄制](https://event.on24.com/wcc/r/3995983/A5FC731C36313CE081AEBB3B1AA39C77)。

<span class="preview">非週期功能（在23.1發佈日期之前發佈到生產部門的功能）以黃色突出顯示。</span>

### 功能即將從Workfront刪除

以下功能即將從Workfront刪除：

#### **從預覽環境中取消和刪除「市場活動（試用版）」區域**

我們將於2023年1月9日從預覽環境中刪除「市場活動（試用版）」區域。 我們的產品團隊將審查此功能並將其納入未來的產品發佈中。

目前，我們正在進行以下更改：

* 從「主菜單」和「預覽」環境中的「佈局」模板中刪除「市場活動（測試版）」區域。
* 從「主菜單」和「生產」環境中的「佈局模板」中，為那些請求啟用了市場活動（測試版）的客戶刪除市場活動(Beta)區域。

此外，描述此功能的所有文章也將被刪除。

#### **清單控制項（從「設定」中的「介面」區域）**

「設定」>「介面」中的「清單控制項」將隨23.1版本一起刪除。

某些清單控制功能正在移至「設定」>「介面」>「過濾器/視圖/分組」區域。 有關詳細資訊，請參見 [建立、編輯和共用預設篩選器、視圖和分組](/help/quicksilver/administration-and-setup/set-up-workfront/configure-system-defaults/create-and-share-default-fvgs.md)。

#### **WorkfrontIllustrator和InDesign**

我們要取代 [WorkfrontIllustrator和InDesign](/help/quicksilver/documents/workfront-for-adobe-creative-cloud/use-wf-adobe-cc.md) 已更新Creative Cloud插件。 從2022年底開始，此擴展將不再可用於在AdobeExchange中安裝。

此擴展中的功能以及幾項增強功能在更新的Adobe Workfront中提供了Creative Cloud插件：
* [Adobe Workfront,Photoshop](/help/quicksilver/workfront-integrations-and-apps/adobe-workfront-for-creative-cloud/wf-cc-install-ps.md)
* [Adobe WorkfrontXD](/help/quicksilver/workfront-integrations-and-apps/adobe-workfront-for-creative-cloud/wf-adobe-xd-install.md)
* [Adobe Workfront供設計和視頻](/help/quicksilver/workfront-integrations-and-apps/adobe-workfront-for-creative-cloud/wf-install-cc.md)

#### **資源計畫程式**

>[!NOTE]
>
>10月20日，資源計畫程式已從預覽環境中刪除。 2023年1月發佈的23.1版將從生產環境中刪除。


作為資源計畫工具棄用的一部分，我們刪除了以下區域：

* 「主菜單」資源區中的「調度」部分

* 團隊頁面的「計畫」部分

* 項目頁面的「計畫」部分

負載平衡器正在替換資源調度程式。
