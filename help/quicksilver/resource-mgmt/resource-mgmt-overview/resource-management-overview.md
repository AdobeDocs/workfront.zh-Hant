# 資源管理

資源管理允許資源管理人員做出業務決策，以確保當前可用的單個員工按照最高優先順序的工作進行預算。

雖然它可能從工作角色級別開始，但最終資源管理會關注用戶、他們分配給實際工作的任務，以及他們的利用率不足或承諾過高。
