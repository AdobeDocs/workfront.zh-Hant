# [!DNL Adobe Workfront] 整合

您可以將Adobe Workfront與協力廠商應用程式以及其他 [!DNL Adobe] 產品。 整合可擴充Workfront的公用程式，並根據貴組織的需求量身打造。

整合 [!DNL Workfront] 分為下列類別：

## 內建（原生）整合

[!DNL Workfront] 提供多種整合功能，您可以直接從Workfront應用程式進行設定，或透過安裝直接從其他應用程式進行設定 [!DNL Workfront] 該應用程式的載入項。

其中部分整合免費提供，而其他整合則需要額外購買：

* [開始使用內建（原生）整合](#get-started-with-built-in-native-integrations)
* [整合可免費使用](#integrations-available-at-no-additional-cost)
* [需要額外購買的整合](#integrations-requiring-an-additional-purchase)

### 開始使用內建（原生）整合 {#get-started-with-built-in-native-integrations}

若要開始使用內建（原生）整合，請遵循下列工作流程：

1. **決定哪些原生整合最符合您的需求**

   選擇整合時，請務必挑選人們已經經常使用的應用程式。 整合 [!DNL Workfront] 使用經常或經常使用的應用程式將提高用戶的採用率。

   如需可用整合的清單，請參閱 [整合可免費使用](#integrations-available-at-no-additional-cost) 和 [需要額外購買的整合](#integrations-requiring-an-additional-purchase).

1. **識別您的技術管理員**

   針對您要與整合的應用程式，識別貴組織的技術管理員 [!DNL Workfront]. 例如，如果您選擇整合 [!DNL Jira]，您必須連線至貴公司的 [!DNL Jira] 管理，並讓他們參與其平台中的任何設定需求。 您也應讓此管理員參與與 [!DNL Workfront] 客戶支援。

   若要開始使用您的管理員，您必須提供他們該整合的特定指示。 如需這些指示的連結，請參閱 [整合可免費使用](#integrations-available-at-no-additional-cost).

1. **參與 [!DNL Workfront] 客戶支援**

   若您的管理員在原生整合設定方面有任何問題，您可以與 [!DNL Workfront] 客戶支援以協助指南或疑難排解。 同時 [!DNL Workfront] 不會為您設定整合，我們承諾為您提供您自行完成整合所需的所有必要工具和資源。 您可以直接與客戶支援人員溝通，或探索詳細的資源文章，協助您輕鬆設定整合。

   若要連線客戶支援，請參閱 [聯絡客戶支援](../../workfront-basics/tips-tricks-and-troubleshooting/contact-customer-support.md).

   如需說明文章的連結，請參閱 [整合可免費使用](#integrations-available-at-no-additional-cost). 和 [需要額外購買的整合](#integrations-requiring-an-additional-purchase).

   有關建立支援案例的說明，請參閱 [聯絡客戶支援](../../workfront-basics/tips-tricks-and-troubleshooting/contact-customer-support.md).

   >[!NOTE]
   >
   >您必須是貴組織的指定支援聯繫人才能提交支援案例。 提交案例後，您可以將另一個系統的技術管理員新增至對話。 他們無法直接與支援人員聯繫。

我們了解，可用的內建（原生）整合可能不符合您公司的所有需求。 如果是這種情況，請連絡您的客戶主管以討論自訂整合。

### 需要額外購買的整合 {#integrations-requiring-an-additional-purchase}

[!DNL Workfront] 可讓您以額外成本運用其他內建的整合和整合產品。 請洽詢您的帳戶代表以取得價格詳細資訊。

* [!DNL Workfront DAM]

   如需詳細資訊，請參閱 [[!DNL Adobe Workfront DAM] with [!DNL Adobe Workfront]](../../documents/workfront-dam-within-workfront/workfront-dam-in-workfrontt.md).

* [!DNL Workfront Fusion]

   如需詳細資訊，請參閱 [[!DNL Adobe Workfront Fusion] 概述](../../workfront-fusion/get-started/workfront-fusion-overview.md).

* [!DNL Adobe Experience Manager Assets]

   如需詳細資訊，請參閱 [設定 [!DNL Workfront] with [!DNL Adobe Experience Manager] 舊連接器](../../administration-and-setup/configure-integrations/configure-workfront-aem.md).

## 合作夥伴整合

[!DNL Workfront] 擁有豐富的合作夥伴生態系統，包含數十種讓您的 [!DNL Workfront] 體驗。

現有合作夥伴產品的範例包括與 [!DNL Salesforce.com], [!DNL Oracle Responsys]，和 [!DNL Microsoft Visual Studio]. 如需這些和其他令人興奮的整合的詳細資訊，請參閱 [[!DNL Workfront] Marketplace](https://www.workfront.com/enterprise/marketplace/apps/).

## [!DNL Workfront] API

此 [!DNL Workfront] API可讓您擴充及增強 [!DNL Workfront] 運用我們的公開API來體驗。 您可以使用我們開放的API建立您自己的整合。 有關 [!DNL Workfront] API，請參閱 [!DNL Workfront] [API Explorer](../../wf-api/general/api-explorer.md).
