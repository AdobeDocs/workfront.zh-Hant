# 變更使用者名稱

您可以編輯使用者的使用者設定檔，以編輯其使用者名稱（電子郵件地址）。 如需指示，請參閱[編輯使用者設定檔](../../../administration-and-setup/add-users/create-and-manage-users/edit-a-users-profile.md)中電子郵件位址列位的相關資訊。
