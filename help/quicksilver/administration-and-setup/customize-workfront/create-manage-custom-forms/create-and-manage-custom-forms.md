# 自訂表單：文章索引

<!-- Audited: 1/2024 -->

您可以建立使用者可附加至Workfront物件的自訂表單。 處理物件的使用者可以填寫自訂表格，以提供物件的相關資訊。

您可以在Workfront執行個體中建立的自訂表單數量沒有限制。 您可以附加至物件的自訂表格數量限製為10個。
