# 允許來自Adobe Workfront應用程式的電子郵件

如果您已將防火牆或郵件伺服器設定為僅允許特定廠商存取，則需要將特定Adobe Workfront IP位址新增至防火牆允許清單。 如需指示，請參閱[設定防火牆的允許清單](../../../administration-and-setup/get-started-wf-administration/configure-your-firewall.md)。
