# 移除各種單一登入選項

Adobe Workfront已停止支援各種單一登入(SSO)選項。 最終可為所有Workfront產品和服務提供一致且更安全的登入體驗。

2020年從產品中刪除了以下SSO選項：

* LDAP
* 主動式目錄
* SAML 1.1

如需可用SSO解決方案的相關資訊，請參閱 [Adobe Workfront中的單一登入概觀](../../add-users/single-sign-on/sso-in-workfront.md).
