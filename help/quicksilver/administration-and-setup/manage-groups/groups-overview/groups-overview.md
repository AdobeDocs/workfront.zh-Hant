# 群組：文章索引

Workfront管理員可建立與您的部門結構一致的使用者群組。 群組與類似，但與團隊和公司不同。

Workfront管理員會授予群組對其需要工作和溝通的Workfront區域的存取權。 然後，每個群組都可以將其Workfront資訊（例如，使用者、範本和自訂表格）和專案與其他部門的資訊分開。
