# 從以下位置訪問收藏夾和最近的項目 [!DNL Slack]

安裝並配置完成後 [!DNL Adobe Workfront for Slack]，您可以檢視 [!UICONTROL Workfront] 收藏夾和最近的項目，並從清單中的任一個訪問項目 [!DNL Slack].

如需有關設定的詳細資訊 [!DNL Workfront with Slack]，請參閱 [設定 [!DNL Adobe Workfront for Slack]](../../workfront-integrations-and-apps/using-workfront-with-slack/configure-workfront-for-slack.md).

## 必要條件

訪問收藏夾和最近的項目之前，請從 [!DNL Slack]，您必須

* 設定 [!DNL Workfront for Slack]\
   配置說明 [!DNL Workfront for Slack]，請參閱 [設定 [!DNL Adobe Workfront for Slack]](../../workfront-integrations-and-apps/using-workfront-with-slack/configure-workfront-for-slack.md).

## 存取 [!UICONTROL 我的最愛] 清單 [!DNL Slack]

1. 登入 [!DNL Slack] 執行個體和登入 [!DNL Workfront] 從 [!DNL Slack].\
   如需登入的詳細資訊，請參閱 [!DNL Workfront] 從 [!DNL Slack]，請參閱「登入」 [!DNL Workfront] 從 [!DNL Slack]「 」部分 [存取 [!DNL Adobe Workfront] 從 [!DNL Slack]](../../workfront-integrations-and-apps/using-workfront-with-slack/access-workfront-from-slack.md).

1. 從任何通道，在 [!UICONTROL 訊息] 欄位： `/workfront favorites`

   >[!NOTE]
   >
   >命令區分大小寫。 您可以使用 `/wf` 而非 `/workfront`.

   隨即顯示您的收藏夾清單。

1. （選用）按一下 **[!UICONTROL 顯示更多]** 以列出更多收藏夾。
1. 按一下收藏的名稱以在中開啟它 [!DNL Workfront] 中的任何值。

## 從訪問最近的項目清單 [!DNL Slack]

1. 登入 [!DNL Slack] 執行個體和登入 [!DNL Workfront] 從Slack。\
   如需登入的詳細資訊，請參閱 [!DNL Workfront] 從 [!DNL Slack]，請參閱「登入」 [!DNL Workfront] 從 [!DNL Slack]「 」部分 [存取 [!DNL Adobe Workfront] 從 [!DNL Slack]](../../workfront-integrations-and-apps/using-workfront-with-slack/access-workfront-from-slack.md).

1. 從任何通道，在 [!UICONTROL 訊息] 欄位： `/workfront recent`

   >[!NOTE]
   >
   >命令區分大小寫。 您可以使用 `/wf` 而非 `/workfront`.

   「最近的項目」清單會依上次存取的順序顯示，最近的項目會顯示在頂端。 項目一次列出三個，並按對象類型分組。\

1. （選用）按一下 **[!UICONTROL 顯示更多]** 以列出更近的項目。
1. （選用）按一下您最近存取的項目名稱，以在中開啟它 [!DNL Workfront] 中的任何值。
