# 開始使用校訂和Creative Cloud Express


我們很高興推出校訂與Creative Cloud Express之間的全新整合。 透過這項整合，您可以

* 使用Workfront核准範本以定義核准工作流程

* 在校訂檢視器中檢閱Creative Cloud Express草稿內容

* 請參閱Creative Cloud Express中的稽核決定，以取得最終核准和發佈

## 先決條件

* Workfront和Creative Cloud Express必須部署至相同的IMS組織。

* 為了讓整合發揮作用，使用者只能在IMS組織內的一個Workfront例項中啟用及作用中

* 需要在unified shell上啟用Workfront執行個體

* Workfront設定區段已切換為連線到校訂（請參閱下文）

## 整合需求
