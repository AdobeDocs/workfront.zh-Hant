# API版本設定和支援排程



Adobe Workfront會定期發行新API版本，通常每年兩次。 API版本在發行後提供3年的支援。 不再支援3年以前的API版本。

如需識別及使用API版本的詳細資訊，請參閱[在您的整合中指定API版本](/help/quicksilver/wf-api/api/specify-api-version-integrations.md)。
