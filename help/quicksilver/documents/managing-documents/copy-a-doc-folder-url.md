# 複製並共用指向文檔資料夾的連結

您可以為下列任何內容中包含的任何文檔資料夾複製直接連結 [!DNL Workfront] 對象： [!UICONTROL 方案], [!UICONTROL Portfolio], [!UICONTROL 專案], [!UICONTROL 任務]，或 [!UICONTROL 問題]. 您無法複製 [!UICONTROL 檔案] 區域，因為這些資料夾直接連結到您的用戶配置檔案，並且無法與其他資料夾共用。

## 複製並共用指向文檔資料夾的連結

要複製指向文檔資料夾的連結，請執行以下操作：

1. 導覽至 [!DNL Workfront] 包含文檔資料夾的對象。
1. 按一下 **[!UICONTROL 更多]** ，然後選取 **[!UICONTROL 複製URL]**. 您可以共用此連結，以快速存取資料夾。 用戶必須至少具有對對象的「查看」訪問權限才能查看資料夾。
   ![複製文檔資料夾url](assets/copy-doc-folder-url.png)
