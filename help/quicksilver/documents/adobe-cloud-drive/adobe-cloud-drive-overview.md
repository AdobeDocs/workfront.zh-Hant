# Adobe Cloud Drive概述

Adobe Cloud Drive是案頭應用程式，可將Adobe雲端儲存空間專案掛載到Mac或Windows電腦上，作為磁碟機。 Workfront專案會顯示為資料夾，您可以使用任何案頭應用程式，直接在Adobe雲端儲存空間開啟、建立、編輯及儲存檔案至Workfront專案。

Adobe Cloud Drive中的檔案與您在Workfront和Frame.io中看到的檔案相同，因此您在案頭上進行的變更會顯示在兩個產品中連線的Workfront專案中。

[觀看概述Adobe Cloud Drive可用功能的影片。](https://video.tv.adobe.com/v/3492492){target="_blank"}

如需Adobe Cloud Drive所連線之存放裝置的詳細資訊，請參閱[Adobe雲端存放裝置概覽](/help/quicksilver/review-and-approve-work/esm-overview.md)。

## 您可以使用Adobe Cloud Drive做什麼

* 在Finder (Mac)或檔案總管(Windows)中，以資料夾形式從Adobe雲端儲存空間存取專案。
* 在Adobe Creative Cloud應用程式、Microsoft Office或電腦上安裝的任何應用程式中開啟、建立及編輯檔案。
* 將變更儲存在本機，並讓Adobe Cloud Drive將其同步至Adobe雲端儲存空間上的專案。
* 在旅行或工作時沒有網路連線時，將檔案標籤為離線存取。 當您的連線恢復時，更新會同步到專案中的檔案，然後您可以從本機快取中移除檔案，以釋放磁碟空間。
* 與相同專案檔案的團隊成員共同作業。

## 誰可以使用Adobe Cloud Drive

若要使用Adobe Cloud Drive，您的組織必須位於已啟用Adobe雲端儲存空間的工作流程Ultimate套件上。

只有使用Adobe雲端儲存空間的專案才會出現在磁碟機中。 舊版Workfront專案不會出現。 如需差異的詳細資訊，請參閱[Adobe雲端儲存空間與舊版Workfront儲存空間之間的差異](/help/quicksilver/review-and-approve-work/esm-overview.md#differences-between-adobe-cloud-storage-and-legacy-workfront-storage)。

如果您沒有看到Adobe Cloud Drive可供下載，或您的組織正在為您管理應用程式安裝，請連絡Workfront管理員。
