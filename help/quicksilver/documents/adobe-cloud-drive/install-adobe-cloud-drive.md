# 安裝Adobe Cloud Drive

若要在案頭上使用Adobe雲端儲存空間專案，請在Mac或Windows電腦上安裝Adobe Cloud Drive。

如需Adobe Cloud Drive是什麼，以及您可以如何操作的詳細資訊，請參閱[Adobe Cloud Drive概觀](/help/quicksilver/documents/adobe-cloud-drive/adobe-cloud-drive-overview.md)。

## 先決條件

在安裝Adobe Cloud Drive之前，請確定以下事項：

* 您的裝置符合本文中[系統需求](#system-requirements)一節所列的系統需求。
* 您的組織使用支援Adobe雲端儲存空間的Workfront版本。

如果貴組織管理應用程式安裝，但您找不到可供下載的Adobe Cloud Drive，請聯絡Workfront管理員。

## 系統需求

**作業系統：**

* macOS 13.5或更新版本（Intel和Apple Silicon）
* Windows 11 22H2 (22621.6060)或更新版本(x64)
* Windows 11 23H2 (22631.6199)或更新版本(ARM)

**硬體：**

* 最少4 GB RAM
* 500 MB的可用磁碟空間供應用程式使用
* 快取檔案的額外空間（視使用狀況而定）

**網路：**

* 用於初始設定和同步的網際網路連線
* 為提升效能，建議使用穩定連線

若要確保Adobe Cloud Drive可以存取其所依賴的服務，請允許[Adobe網路端點](https://helpx.adobe.com/in/enterprise/kb/network-endpoints.html)中列出的網路端點。

## 下載安裝程式

1. 移至[Adobe軟體發佈](https://experience.adobe.com/#/downloads)。

1. 使用您的 Adobe ID 登入。
1. 在&#x200B;**一般**&#x200B;下載清單中，找到&#x200B;**Adobe雲端磁碟機**。
1. 選取作業系統的安裝程式：

   * **macOS：**&#x200B;適用於Mac （ARM64或Intel）的Adobe Cloud Drive
   * **Windows：**&#x200B;適用於Windows （x64或ARM64）的Adobe Cloud Drive

1. 按一下「**下載**」。

   >[!IMPORTANT]
   >
   >如果您沒有看到Adobe Cloud Drive安裝程式，表示您的組織可能未使用支援Adobe雲端儲存空間的Workfront版本。 請連絡您的Workfront管理員。

## 安裝Adobe Cloud Drive

請使用作業系統的程式。

### 在macOS上安裝

1. 開啟下載的`.dmg`檔案。
1. 連按兩下顯示的安裝圖示。
1. 按照熒幕上的設定精靈進行。

1. 出現提示時，請使用您的Adobe ID登入。

   如果另一個Adobe案頭應用程式已在執行中，而且您已登入，Adobe Cloud Drive會自動使用該作用中工作階段。

### 在Windows上安裝

1. 執行下載的`.exe`安裝程式。

1. 按照安裝精靈中的提示操作。
1. 出現提示時，按一下&#x200B;**安裝**。
1. 從[開始]功能表啟動&#x200B;**Adobe Cloud Drive**。
1. 出現提示時，請使用您的Adobe ID登入。

## 驗證安裝

安裝後，Adobe Cloud Drive會以已掛載磁碟機的形式顯示在電腦上：

* **macOS：**&#x200B;在&#x200B;**位置**&#x200B;下的Finder側邊欄中，做為&#x200B;**Adobe雲端磁碟機**。
* **Windows：**&#x200B;在&#x200B;**這部電腦**&#x200B;下的檔案總管中，做為&#x200B;**Adobe雲端磁碟機**。

您應該會在磁碟機中看到您的Adobe雲端儲存空間專案被列為資料夾。

如需在磁碟機中使用專案的詳細資訊，請參閱[使用Adobe雲端磁碟機](/help/quicksilver/documents/adobe-cloud-drive/use-adobe-cloud-drive.md)。
