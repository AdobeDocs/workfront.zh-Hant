# 中的資料結構 [!DNL Adobe Workfront Fusion]

資料結構是一種文檔，詳細說明要傳輸到的資料的格式 [!DNL Adobe Workfront Fusion]. 根據本文檔，方案編輯器能夠確定返回或接收哪種資料的模組。 資料結構檔案最常用於序列化/剖析資料格式，例如JSON、XML、CSV等。

您可以按一下 [!UICONTROL 建立新資料結構] 按鈕 [!UICONTROL 資料結構概觀] 區段或需要資料結構規範的模組設定中。

支援的資料類型在 [[!UICONTROL 模組類型]](../../workfront-fusion/modules/module-types.md) 文章。

## 資料結構產生器

資料結構不一定都需要建立。 使用內建產生器的範本，可讓操作更輕鬆。 提供資料範例後，產生器就會根據您輸入的資料範例自動建立資料結構。 然後，可手動修改建立的資料結構。
