# 選擇觸發程式模組的開始位置 [!DNL Adobe Workfront Fusion]

有些觸發程式模組可讓您選取要開始擷取套裝的第一個套裝。

您也可以指定是否要從特定日期後擷取所有套件組合，或僅擷取套件組合。

如需觸發程式模組的詳細資訊，請參閱區段 [觸發器模組](../../workfront-fusion/modules/module-types.md#triggers) 在文章中 [模組型別](../../workfront-fusion/modules/module-types.md).

## 選擇觸發程式模組的開始位置

1. 儲存觸發程式模組。

   或

   變更觸發程式模組的設定，如所述 [在中設定模組的設定 [!UICONTROL Adobe Workfront Fusion]](../../workfront-fusion/modules/configure-a-modules-settings.md).

   或

   以滑鼠右鍵按一下中觸發程式模組的圖示 [!UICONTROL 案例編輯器]，如所述 [在中建立情境 [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/create-a-scenario.md).

1. 選取「 」中的「 」選項 **[!UICONTROL 選擇開始位置]** 方塊中顯示的方塊。

   ![](assets/choose-where-to-start-350x346.jpg)

   顯示的選項視特定服務的可能性而定。 其中可能包括：

   <table style="table-layout:auto">
        <tr>
            <td>[！UICONTROL從現在開始] （預設）</td>
            <td>從現在起擷取新增或更新的所有組合（視設定而定）</td>
        </tr>
        <tr>
            <td>[！UICONTROL從特定日期之後]</td>
            <td>擷取在指定日期/時間後新增或更新的所有組合（視設定而定）</td>
        </tr>
        <tr>
            <td>[！UICONTROL ID大於或等於特定值]</td>
            <td>擷取ID大於或等於指定ID的所有組合</td> 
        </tr>
        <tr>
            <td>[！UICONTROL所有組合]</td>
            <td>擷取所有可用的組合</td>
        </tr>
        <tr>
            <td>[！UICONTROL選取第一個組合]</td>
            <td>可讓您選取要開始擷取束的第一個束</td>
        </tr>
   </table>
