# HTTP模組

>[!NOTE]
>
>[!DNL Adobe Workfront Fusion] 要求 [!DNL Adobe Workfront Fusion] 除 [!DNL Adobe Workfront license].

HTTP應用程式提供了基於超文本傳輸協定(HTTP)協定的各種通信模組。 HTTP是萬維網資料通信的基礎。 您可以使用模組來下載網頁和檔案、呼叫Webhook和API端點等。

模組的正確選擇取決於要訪問的資源所採用的身份驗證/授權機制。
