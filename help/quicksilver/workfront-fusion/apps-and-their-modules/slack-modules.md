# [!DNL Slack] 模組

在 [!DNL Adobe Workfront Fusion] 案例中，您可以自動執行使用 [!DNL Slack]，並將其連接至多個協力廠商應用程式和服務。

如果您需要建立案例的相關指示，請參閱 [在中建立案例 [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/create-a-scenario.md).

如需模組的相關資訊，請參閱 [中的模組 [!DNL Adobe Workfront Fusion]](../../workfront-fusion/modules/modules.md).

## [!DNL Slack] 模組及其欄位

## 術語

下列術語在設定 [!DNL Slack] 模組：

* **DM**: [!UICONTROL 直接訊息]
* **IM**: [!UICONTROL 即時消息]
* **私人管道**:formely [!UICONTROL 群組]

* **管道**: [!UICONTROL 對話] 在API檔案中， [!UICONTROL 頻道] 在 [!DNL Slack] 應用程式。
