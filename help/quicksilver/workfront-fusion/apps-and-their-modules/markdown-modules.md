# [!UICONTROL Markdown] 模組

在 [!DNL Adobe Workfront Fusion] 案例中，您可以使用 [!UICONTROL Markdown] 模組，將Markdown轉換為HTML，並將HTML轉換為Markdown。

如果您需要建立案例的相關指示，請參閱 [在中建立案例 [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/create-a-scenario.md).

如需模組的相關資訊，請參閱 [中的模組 [!DNL Adobe Workfront Fusion]](../../workfront-fusion/modules/modules.md).

## [!UICONTROL Markdown至HTML]

此模組會將Markdown轉換為HTML。

設定此模組時，會顯示下列欄位。

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td role="rowheader">[!UICONTROL Markdown]</td> 
   <td> <p>輸入Markdown格式的純文字。</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!DNL GitHub Flavored Markdown] </td> 
   <td> <p>啟用此選項，將GitHub Flavored Markdown轉換為HTML。</p> <p>如需詳細資訊，請參閱3月[!DNL ]查了查查表 [!DNL GitHub] 檔案。</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!UICONTROL處理]</td> 
   <td>選取選項，以指出您要從文字中移除HTML標籤，還是逸出HTML。</td> 
  </tr> 
 </tbody> 
</table>

## [!UICONTROL HTML至Markdown]

此模組會將HTML程式碼轉換為Markdown。

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td role="rowheader">[!UICONTROL Markdown]</td>

  </tr> 
  <tr> 
   <td role="rowheader">[!DNL GitHub Flavored Markdown] </td> 
   <td> <p>啟用此選項可將HTML轉換為 [!DNL GitHub Flavored Markdown].</p> <p>如需詳細資訊，請參閱 [!DNL GitHub] 檔案。</p> </td> 
  </tr>
