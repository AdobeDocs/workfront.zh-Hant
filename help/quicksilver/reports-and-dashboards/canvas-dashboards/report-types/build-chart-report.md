# 在畫布儀表板中建立圖表報告

可將您的資料視覺化為橫條、欄、折線圖或圓餅圖的圖表報表，可新增至畫布控制面板。

如需在畫布控制面板上新增和排列報表的一般資訊，請參閱[在畫布控制面板中新增、移除或排列報表](/help/quicksilver/reports-and-dashboards/canvas-dashboards/manage-canvas-dashboards/add-remove-arrange-reports.md)。
